Measuring Feature Importance

The feature importance ranking established in enpls are based on the average estimated effect size of variables and the variance of coefficients. The ranking serves as an informative measurement for the contribution of each variable in the (sparse) PLS model.

In enpls, the function enpls.fs() computes the feature importance ranking based on ensemble PLS regressions while enspls.fs() gives a similar ranking based on ensemble sparse PLS regressions.

Outlier Detection





When some samples in the dataset are selected as the out-of-bag samples in the base PLS models, the mean and variance of the prediction errors on each individual sample could reflect the fact if the sample is a potential outlier.

The functions enpls.od() and enspls.od() in enpls detect outliers based on the above idea.

Applicability Domain Evaluation





Model applicability domain measures how well the PLS/SPLS model built on the training set performs on external test sets. In enpls, after a certain perturbation was applied to the variables or samples of the training set, the error distribution derived from the predictions of all sub-models could characterize this domain.

The functions enpls.ad() and enspls.ad() in enpls could evaluate the model applicability domain.

Fitting Ensemble (Sparse) PLS Models





The (sparse) partial least squares regression model is used as the base learner in enpls. We build many different models with bootstrapped or Monte-Carlo resampled training sets. The models are then aggregated to establish an ensemble (S)PLS model.

The functions enpls.fit() and enspls.fit() support fitting the ensemble models; cv.enpls() and cv.enspls() support cross-validation on such ensemble models.

about enpls

The idea of ensemble learning is not only effective in improving the predictive performance of certain statistical and machine learning models, but also useful for model diagnostic purposes. In the light of this, we built the enpls package, which offers a framework for measuring feature importance, outlier detection, and ensemble modeling with (sparse) partial least squares regressions.

If you have any questions, suggestions, or ideas about the package, please feel free to let me know via email or create an issue on GitHub.
